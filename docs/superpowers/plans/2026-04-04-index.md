## Prerequisites

Before starting Sprint 1:
- [ ] MySQL running locally with database `intelligent_job_matching` created
- [ ] Python venv working (`pip install -r requirements.txt`)
- [ ] Node.js 18+ installed (`cd frontend && npm install`)
- [ ] SWI-Prolog downloaded (install in Sprint 2 Task 2.1)

## Key Decisions

- **Prolog:** pyswip (Python bindings to SWI-Prolog). Rules in `matching.pl`.
- **Auth:** Token stays as `token_{email}_{timestamp}` — no JWT migration.
- **DB:** MySQL via `mysql+pymysql`. Config from `.env`.
- **File uploads:** Local `uploads/` directory. Served by Flask at `/api/uploads/`.
- **Messaging poll:** 5s interval for thread, 30s for unread badge.
- **UI style:** Style B — white background, indigo (#6366f1) accent, Inter font.
